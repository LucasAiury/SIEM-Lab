# SIEM-Lab
Este laboratório foi desenvolvido com o objetivo de simular um ambiente básico de monitoramento de segurança utilizando Microsoft Defender em Microsoft Azure.
____________________________________________________________________________

O alerta analisado envolveu:
- Tentativas de login em contas desativadas
- Múltiplas falhas de autenticação
- Atividade suspeita originada de um mesmo endereço IP
____________________________________________________________________________

Objetivo
- Identificação de atividades suspeitas
- Análise inicial de alertas de segurança
- Desenvolvimento de familiaridade com o Microsoft Defender
- Criação de base para resposta a incidentes em ambiente SIEM
