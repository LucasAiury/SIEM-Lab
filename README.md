# SIEM-Lab
Este laboratório foi criado com o objetivo de simular um ambiente básico de monitoramento de segurança com Microsoft Defender em Microsoft Azure.

Etapas:
- Configuração inicial do ambiente
- Criação de conta no Microsoft Azure
- Acesso ao portal Azure
- Criação de Resource Groups e utilização de dados de demonstração de alertas de segurança
- Ativação do Microsoft Sentinel
- Ativação do Microsoft Defender
- Habilitação de recursos de monitoramento e segurança
- Análise de alertas e recomendações associadas
____________________________________________________________________________
O alerta em foco envolveu:
- Tentativas de login em contas desativadas
- Múltiplas falhas de autenticação
- Atividade suspeita originada de um mesmo IP
____________________________________________________________________________

O objetivo é identificação de atividades suspeitas, análise inicial de alertas de segurança, desenvolver familiaridade com o MS Defender e criar uma base para resposta a incidentes em ambiente SIEM.
